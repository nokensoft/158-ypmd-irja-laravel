# YPMD IRJA — Website Resmi

Website statis untuk **Yayasan Pengembangan Masyarakat Desa Irian Jaya (YPMD IRJA)**, LSM pertama di Tanah Papua yang berdiri sejak 1984.

🔗 **Demo:** [https://nokensoft.github.io/158-ypmd-irja/](https://nokensoft.github.io/158-ypmd-irja/)

## Fitur Utama

- **10 Halaman** — Beranda, Sejarah, Profil, Tokoh Kunci, Galeri, Program, KDK (Buletin), Papua Today (Blog), Donasi, Kontak
- **Responsive Design** — Tampilan optimal di desktop, tablet, dan mobile
- **Navigasi Dropdown & Accordion** — Menu desktop hover dropdown, mobile accordion (Alpine.js)
- **Timeline Interaktif** — Sejarah & tonggak penting YPMD IRJA dengan chart (Chart.js)
- **Arsip Buletin KDK** — Kabar Dari Kampung dengan sidebar filter bulan/tahun & pagination
- **Papua Today** — Blog dengan sidebar kategori & pagination
- **Galeri Foto** — Grid gambar dengan efek hover zoom
- **Halaman Donasi** — Informasi rekening, galeri dampak kegiatan, konfirmasi via WhatsApp
- **Google Maps** — Peta lokasi kantor di halaman Kontak
- **Footer** — Navigasi, media sosial (IG, FB, YouTube, WA), FAQ, Disclaimer

## Tech Stack

- HTML5 + Tailwind CSS (CDN)
- Alpine.js (interaktif)
- Chart.js (visualisasi data)
- Font Awesome (ikon)
- Google Fonts (Lora + Plus Jakarta Sans)
